
**************************************************************
Настройка конфигурации IDE Borland Delphi 7
**************************************************************

Внешний вид панелей среды разработки для разрешения экрана
1280x1024 и файлы с настройками размещения окон для обычного
и отладочного режимов. DST-файлы должны быть помещены в
каталог C:\Program Files\Borland\Delphi7\Bin, чтобы заменить
собой одноименные файлы, созданные средой.

Для удобства работы с окном Code Explorer можно пометить
следующие категории в его настройках: Fields, Properties,
Methods, Variables/Constants, Uses, Inherited, а Explorer
Sorting установить в положение "Source".
